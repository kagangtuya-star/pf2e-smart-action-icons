[English README](./README.md)

这是一个用于 Pathfinder 2e 的 Foundry VTT v13 模组，根据动作类型自动修正 PF2E 的动作/专长物品图标。

## 功能

- 提供两个世界设置（默认都开启）：
  - `替换 NPC 物品图标`
  - `替换物品图标`
- 仅在 PF2E 系统下运行。

## 兼容性

- Foundry VTT：v13
- 游戏系统：仅 PF2E

## 配置

打开 `游戏设置 -> 配置设置 -> 模组设置 -> PF2E Smart Action Icons`。

可用设置：

- `替换 NPC 物品图标`：控制 NPC 角色所持有物品的图标替换。
- `替换物品图标`：控制世界物品与非 NPC 持有物品的图标替换。
