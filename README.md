# PF2E Smart Action Icons

[简体中文说明](./README.zh-CN.md)

Foundry VTT v13 module for Pathfinder 2e.Auto-corrects PF2E action/feat item icons based on action type.

## Features

- Runs automatically in the background after world load.
- Two world settings (both enabled by default):
  - `Replace NPC Item Icons`
  - `Replace Item Icons`
- Restricted to PF2E system only.

## Compatibility

- Foundry VTT: v13
- Game System: PF2E only

## Configuration

Open `Game Settings -> Configure Settings -> Module Settings -> PF2E Smart Action Icons`.

Available settings:

- `Replace NPC Item Icons`: Controls replacement for items owned by NPC actors.
- `Replace Item Icons`: Controls replacement for world items and non-NPC owned items.
